Type            : Quake 2 player model + skin
Name            : Garfield
Author          : Lars Jensen
Time to make    : 3 days intense work
used to make    : Qukae 2 model editor v0.83

installation process:
To install this awesome player model for Q2, just make a DIR under your
quake2\baseq2\players DIR called Garfield and unzip the files into this DIR.
  Execute quake2 and go to the multiplayer menu. Then select player setup
and choose garfield under model. Ready to kill. Have a good time.
